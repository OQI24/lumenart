# LumenArt

Лендинг компании **LumenArt** — индивидуальный дизайнерский свет, изготовление и монтаж под ключ.

## Стек

- Next.js (App Router, static export)
- TypeScript
- Tailwind CSS
- React Hook Form

## Команды

```bash
npm install
npm run dev      # локальная разработка
npm run build    # сборка статики в out/
npm run lint     # проверка ESLint
```
